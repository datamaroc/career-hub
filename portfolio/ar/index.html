الخطوة 2: نقل الكود البرمجي 💻✨

حان الوقت لاستدعاء ونسخ مشروعك من GitHub إلى جهازك:

🔁 استنساخ المشروع (Repository):

git clone https://github.com/datamaroc/portfolio-starter-kit.git

📁 انتقل إلى مجلد المشروع:

cd portfolio-starter-kit

📦 تثبيت المكونات :

npm install

💡 هذا الأمر يقرأ ملف package.json الخاص بك ويقوم بتنزيل كل ما يجعل موقعك يبدو رائعًا. الأمر أشبه بالطهي، ولكن باستخدام الأكواد.

الخطوة 3: سحر رسائل البريد الإلكتروني مع Supabase 📧

هل تريد أن يتمكن زوارك من مراسلتك بدلاً من التحديق في معرض أعمالك فقط؟ لنوظف قوة الباك-إند مع Supabase – إنها مثل Firebase، ولكن بروح أبسط وأروع ومع SQL.

🔐 1. التسجيل في Supabase

اذهب إلى supabase.com وأنشئ حسابًا. نعم، إنه مجاني. ونعم، إنه رائع.

🏗️ 2. إنشاء مشروع جديد

  • Create New Project <- Create New organization
  • أطلق عليه اسم "My Portfolio"
  • عيّن كلمة مرور قوية.

🔑 3. احصل على مفاتيح API الخاصة بك

اذهب إلى Settings in your project > Data API. وانسخ:

  • Project URL
  • "Go to API Keys" > "anon public key"

🧾 4. إنشاء ملف env.

داخل المجلد الرئيسي لمشروعك، أنشئ ملفًا باسم env. : 

VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key

📝 استبدل القيم بمفاتيحك الحقيقية.

📬 5. إنشاء جدول الرسائل (messages)

في لوحة تحكم Supabase:

  • اذهب إلى Table Editor
  • انقر على New Table، وسمّه messages
  • أضف 4 أعمدة:(columns) [name, email, subject, message] – كلها من نوع نصي (text)

📨 هذا هو المكان الذي سيترك فيه معجبوك رسائلهم.

الخطوة 5: خصّص البورتفوليو الخاص بك 🎨

هنا يحدث السحر الحقيقي. 🪄

أضف شخصيتك ومهاراتك وأسلوبك إلى معرض أعمالك — إنها قصتك، مروية بالكود.

📂 حدّث معلوماتك في src/data

  • user.ts: الاسم، المسمى الوظيفي، البريد الإلكتروني، الشبكات الاجتماعية
  • projects.ts: اعرض أعمالك المذهلة
  • skills.ts: حدّث الأدوات والشهادات
  • education.ts: استعرض مسيرتك التعليمية
  • services.ts: ما الذي تقدمه للعملاء

💡 نصيحة احترافية: انسخ محتوى هذه الملفات مع سيرتك الذاتية واطلب من ChatGPT أو Gemini ترتيب كل شيء لك — كمساعدك الرقمي الخاص.

🖼️ حدّث صورك

  • public/images/profile.png: استبدلها بصورتك الشخصية
  • public/projects/: أضف صور مشاريعك
  • طابق أسماء الملفات مع تلك المستخدمة في projects.ts

📝 حدّث بيانات HTML الوصفية

افتح index.html وحدّث:

  • <title> — اسمك الكامل أو منصبك
  • <meta name="description"> — ملخص قصير عنك أو عن علامتك التجارية
خطوة إضافية: اسمح للآخرين بحجز اجتماع معك 🗓️

اجعل الأمر سهلاً للغاية على العملاء المحتملين أو أصحاب العمل لجدولة اجتماع معك. سنستخدم Calendly لحجز المواعيد بشكل مجاني وسهل.

إعداد رابط Calendly الخاص بك

  1. اذهب إلى calendly.com وسجل للحصول على حساب مجاني.
  2. اتبع الإعدادات لإنشاء أول نوع حدث لك، مثل "اجتماع لمدة 30 دقيقة".
  3. ابحث عن رابط Calendly الشخصي وانسخه. سيبدو هكذا:
https://calendly.com/your-username/30min

💡 أضف هذا الرابط إلى "src/data/user.ts" في معرض أعمالك.

الخطوة 6: انشر موقعك للعالم 🌍

هيا نطلق موقعك إلى عالم الإنترنت — حتى يتمكن عملاؤك وأصحاب العمل المستقبليون من العثور عليك.

سنستخدم GitHub لتخزين الكود و Vercel لاستضافته... مجانًا

🚀 ارفع الكود إلى GitHub

  1. أنشئ مستودعًا جديدًا (Repository): اذهب إلى github.com وأنشئ مستودعًا فارغًا. لا تقم بإضافة ملف README أو .gitignore.
  2. أزل سجل Git القديم (إعادة تعيين اختيارية): 
    rm -rf .git
# على ويندوز:
rd /s /q .git
  3. ابدأ وارفع الكود: 
    git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/your-username/your-repo.git
git push -u origin main

    🧠 استبدل your-username و your-repo بمعلوماتك الحقيقية.

☁️ استضف الموقع على Vercel

  1. اذهب إلى vercel.com وسجّل الدخول باستخدام حساب GitHub.
  2. من لوحة تحكم Vercel، انقر على "Add New ➜ Project".
  3. اختر مستودع البورتفوليو الخاص بك وانقر على Import.
  4. أضف Environment Variables:

    قبل النشر، قم بتوسيع قسم "Environment Variables". يجب عليك نسخ المفاتيح من ملف .env المحلي.

    • أضف VITE_SUPABASE_URL وقيمته.
    • أضف VITE_SUPABASE_ANON_KEY وقيمته.

    🔒 هذه الخطوة تربط موقعك المباشر بمنصة Supabase بشكل آمن لكي يعمل نموذج الاتصال.

  5. سيقوم Vercel باكتشاف Vite تلقائيًا. الآن، فقط انقر على Deploy. فوز سهل

🎉 بمجرد النشر، سيكون موقعك متاحًا على yourproject.vercel.app. حان وقت الاحتفال

الخطوة 7 (اختياري): أضف دومين خاص 🌐

هل تريد أن يكون عنوان موقعك yourname.com؟ لنجعل الأمر رسميًا.

هذه الخطوة اختيارية تمامًا، ولكنها تمنح موقعك لمسة احترافية 💼

  1. اشترِ دومين (نطاق) من مسجّل مثل Namecheap أو GoDaddy.
  2. في Vercel، اذهب إلى إعدادات مشروعك ← تبويب Domains.
  3. انقر على Add واكتب الدومين الخاص بك (مثال: yourname.com).
  4. اتبع تعليمات Vercel لتحديث سجلات DNS لدى المسجّل الخاص بك.

🌍 سيعيش معرض أعمالك الآن على الدومين الخاص بك — أنيق، احترافي، ولا يُنسى.
